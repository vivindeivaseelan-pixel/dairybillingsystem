Full Dairy System

Billing, inventory, customers, and reports in one upgraded project.

The original billing page has been converted into a proper multi-part dairy management workspace using Node.js, React, Python, HTML, CSS, Bootstrap 5, and JavaScript.

Node API React Frontend Python Analytics Bootstrap 5 UI

Project folders

  • frontend/ React dashboard and billing UI
  • backend/ Express API and shared JSON data store
  • python/ Reporting and analytics script
  • README.md Setup and run instructions

Dashboard

Revenue cards, recent invoices, top products, and stock health.

Billing

Create invoices with customers, payment mode, delivery slot, and totals.

Customers & Products

Manage customer records and maintain a live dairy product catalog.

Python Reports

Generate quick analytics from the same stored invoices and customer data.

How to run

cd backend
npm install
npm run dev 

cd frontend
npm install
npm run dev

cd python
python analytics.py
